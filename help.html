Get Started

1

Create an account

Sign in to sync your lists across devices. Guest mode uses local storage only.

2

Create a list

Go to Create List and add a new list name. It will appear in Your Lists.

3

Add tasks and reminders

Open a list, add tasks, and set a due time for reminders and notifications.

Sync and Notifications

Sync status

If you are signed in, tasks sync through Firebase. If you are a guest, tasks stay on this device.

Notifications

Allow notifications when prompted. On iOS Safari, OneSignal handles alerts.

Upcoming tasks

The Upcoming Tasks section shows tasks with due times. Add a due time to see them there.

Troubleshooting

Lists not syncing

Check that you are signed in and refresh the page. Guest mode does not sync.

Notifications not firing

Confirm browser permissions and that a due time is set on the task.

Need more help?

Open Options to adjust reminders, voice, and notification settings.
